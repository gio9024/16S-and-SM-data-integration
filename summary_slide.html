16S & Shotgun Metagenomics
Integration — Project Summary

Three parallel pipelines benchmarked across two QC datasets, comparing 16S amplicon and whole-genome shotgun classification with harmonized taxonomy strategies.

Three Analysis Pipelines
Pipeline 1

Greengenes2 + Woltka

16S DADA2 → GG2 NB Classifier → Genus Table
MGS Bowtie2 / WoLr2 → Woltka OGU → Genus Table

Key Advantage: Both 16S and MGS sides share the same Greengenes2 (2024.09) taxonomy tree — enabling direct label-level comparison with no additional harmonization.

Pipeline 2

RefSeq + Kraken2/Bracken

16S DADA2 → RefSeq V4 Classifier → Genus Table
MGS Kraken2 (full RefSeq) → Bracken → Genus Table

Key Advantage: Full RefSeq coverage on both sides uses NCBI taxonomy. Kraken2 + Bracken provides fast, accurate k-mer–based classification with lowest FP rate among MGS methods.

Pipeline 3

SortMeRNA → Kraken2/Bracken

16S DADA2 → RefSeq V4 Classifier → Genus Table
MGS SortMeRNA (16S extract) → Kraken2 + Bracken → Genus Table

Key Advantage: Extracts only 16S rRNA reads from WGS data before classification, isolating the same marker gene carried by amplicon sequencing for maximum conceptual parity.

Reference Databases
🌿
Greengenes2
v2024.09 · GTDB taxonomy
Unified phylogenomic reference integrating full-length 16S with whole-genome data. Single taxonomy tree usable by both QIIME2 and Woltka.
P1 16S P1 MGS (taxonomy)
🌐
WoLr2 (Web of Life r2)
Bowtie2 index · Prokaryotic reference genomes
Comprehensive prokaryotic genome database for WGS read alignment. Shotgun reads mapped via Bowtie2, then classified by Woltka using GG2 taxonomy.
P1 MGS (alignment)
🧬
NCBI RefSeq (full genome)
Build 2024-01-12 · NCBI taxonomy
Comprehensive curated genomic database. Kraken2/Bracken standard index (72 GB hash table) covering bacteria, archaea, fungi, viruses, and human.
P2 MGS P3 MGS
🔬
NCBI 16S RefSeq
Nov 2024 · 26,244 dereplicated sequences
Curated 16S rRNA sequences from NCBI RefSeq. V4-region NB classifier for amplicons; full-length NB classifier for SortMeRNA-extracted reads.
P2 16S P3 16S P3 MGS
🎯
SortMeRNA rRNA (SILVA)
v4.3 · SILVA-based rRNA reference
SILVA-derived rRNA reference shipped with SortMeRNA. Used to identify and extract 16S reads from raw shotgun FASTQs (~1.5% of reads recovered).
P3 MGS extraction
