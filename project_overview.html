Meta-Analysis Framework

A multi-pipeline architecture for integrated microbiome analysis, delivering high comparability across 16S amplicon and shotgun metagenomics.

🎯

Primary Objective

Unified taxonomic and functional profiling to bridge the gap between sequencing methodologies.

PIPELINE 01

Greengenes2 Integration

  • Unified TaxonomyShared GG2 tree for 16S and shotgun outputs.
  • 16S: QIIME2 & DADA2Classified against GG2 2024.09 backbone.
  • Shotgun: WoltkaWoLr2 alignments mapped directly to GG2 OGUs.
PIPELINE 02

NCBI RefSeq Standard

  • Comprehensive ScopeLeveraging the depth of the NCBI RefSeq database.
  • 16S: RefSeq ClassifierStandard QIIME2 workflow with NCBI taxonomy.
  • Shotgun: Kraken2 & BrackenFast k-mer classification via MOSHPIT framework.
PIPELINE 03

16S Feature Extraction

  • Marker Gene IsolationExtracting 16S reads from MGS for identical processing.
  • Tool: SortMeRNABioinformatic isolation of rRNA reads from pools.
  • Perfect Parity16S and Shotgun run through the same NB classifier.
