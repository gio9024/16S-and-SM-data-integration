Three-Pipeline Full-Sample Report (Portable)

Contents
- interactive_pipeline_report.html : merged interactive report
- html-assets/ : SVG plots used by the interactive report
- results/ : individual pipeline PDFs + combined comparison PDF
- combined_stats.tsv : tabular metrics used to build the report

How to use
1) Keep this folder structure unchanged.
2) Open interactive_pipeline_report.html in any modern browser.
3) If PDF links do not open on double-click in your environment, open files manually from results/.

This package is self-contained for local sharing.
